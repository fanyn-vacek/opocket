CZ:

O:Pocket

O:Pocket je přehledný informační systém pro organizátory orientačního běhu, který klade důraz na jednoduchost a okamžitou dostupnost výsledků.


Hlavní přínosy systému:

Vše na jednom místě: Od registrace uživatele až po historické žebříčky.

Dynamické výsledky: Okamžitý výpočet pořadí v kategorii bez nutnosti ručního třídění.

Integrovaná správa: Smazáním kategorie systém automaticky vyřeší odhlášení všech účastníků, čímž předchází chybám v datech.

Lehkost: Díky SQLite databázi je systém nenáročný na zdroje a snadno přenositelný.


Technické specifikace:

Jazyk: Java 17 (Spring Boot 3).

Frontend: HTML5, CSS3, Thymeleaf.

Databáze: SQLite.


EN:

O:Pocket

O:Pocket is a clear information system for orienteering organizers that emphasizes simplicity and immediate availability of results.


Main benefits of the system:

Everything in one place: From user registration to historical rankings.

Dynamic results: Immediate calculation of rank within a category without the need for manual sorting.

Integrated management: By deleting a category, the system automatically handles the unregistration of all participants, thereby preventing data errors.

Lightweight: Thanks to the SQLite database, the system is resource-efficient and easily portable.


Technical specifications:

Language: Java 17 (Spring Boot 3).

Frontend: HTML5, CSS3, Thymeleaf.

Database: SQLite.
